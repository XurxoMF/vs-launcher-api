# What is this?

For the VS Launcher to work properly I needed a way to get changelog updates, news, versions, files and many other things.

Where is the problem? Anego Studios does not have any API for this. There is one for the ModDB for not for this.

What was my solution, creating a little API to serve all this things in one palce.

### Why the .zip files with the game?

Anego Studios does not serve .zip files with the windows game content so I had no other option than making my own ones.

Those are literally the game downloaded from the official website, installed and compressed... I didn't modified anything... It's just to make it easier for you as user and for me as dev to implement all the auto-download and installation of the game.
